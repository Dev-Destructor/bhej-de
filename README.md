# bhej-de

A full-stack file sharing app

<br/><br/>

## Dependencies
```bash
react.js
next.js
tailwindCss
daisyUI
react-toastify
express
mongoose
multer
nodemailer
cors
helmet
brcypt
```
<br/><br/>

## How to use
1. Go to the server directory
2. Run the following command
> Sudo docker-compose up
